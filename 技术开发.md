# 技术架构与开发指南

## 1. 技术栈选择

| 技术 | 用途 | 选择理由 |
|------|------|----------|
| **HTML5** | 页面结构 | 语义化标签，良好的 SEO 和可访问性。 |
| **CSS3** | 样式与动画 | 实现全部视觉效果和交互，不依赖 JavaScript。 |
| **Flexbox** | 一维布局 | 导航栏、卡片列表等线性布局的首选方案。 |
| **CSS Grid** | 二维布局 | 页面整体框架、图文混排等复杂布局。 |
| **CSS 变量** | 主题管理 | 统一颜色和尺寸，便于主题切换和响应式适配。 |

## 2. 本地运行步骤

```bash
# 1. 进入项目目录
cd 个人主页展示（二）展示

# 2. 使用浏览器直接打开入口文件
start index.html
# 或者双击 index.html 文件

# 3. 开发建议：使用 Live Server 插件预览
# - VS Code 安装 "Live Server" 插件
# - 右键 index.html 选择 "Open with Live Server"
```

## 3. 核心样式模块

### 3.1 CSS 变量定义

```css
:root {
    /* 颜色变量 */
    --color-primary: #e8b86d;      /* 温暖黄色 */
    --color-secondary: #8ecae6;    /* 清爽蓝色 */
    --color-accent: #ffb7b2;       /* 珊瑚粉色 */
    --color-text: #333333;
    --color-text-light: #666666;
    --color-background: #faf8f5;
    --color-white: #ffffff;
    
    /* 字号变量 */
    --font-size-base: 1rem;
    --font-size-lg: 1.125rem;
    --font-size-xl: 1.5rem;
    --font-size-2xl: 2rem;
    
    /* 间距变量 */
    --spacing-sm: 0.5rem;
    --spacing-md: 1rem;
    --spacing-lg: 1.5rem;
    --spacing-xl: 2rem;
    
    /* 圆角变量 */
    --border-radius-base: 0.5rem;
    --border-radius-lg: 0.75rem;
}
```

### 3.2 常用组件样式

**按钮组件**

```css
.button {
    display: inline-flex;
    align-items: center;
    justify-content: center;
    padding: var(--spacing-sm) var(--spacing-lg);
    border-radius: var(--border-radius-base);
    transition: all 0.2s ease;
}

.button--primary {
    background-color: var(--color-primary);
    color: var(--color-white);
}

.enter-button {
    padding: 1rem 3rem;
    background: linear-gradient(135deg, var(--color-primary), var(--color-accent));
    border-radius: 50px;
    box-shadow: 0 4px 15px rgba(232, 184, 109, 0.4);
    transition: all 0.3s ease;
}

.enter-button:hover {
    transform: translateY(-3px) scale(1.05);
}
```

**卡片组件**

```css
.card {
    background-color: var(--color-white);
    border-radius: var(--border-radius-lg);
    box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);
    transition: transform 0.3s ease, box-shadow 0.3s ease;
}

.card:hover {
    transform: translateY(-5px);
    box-shadow: 0 10px 15px rgba(0, 0, 0, 0.1);
}
```

**导航组件**

```css

    }
```

## 4. 响应式断点

| 断点 | 屏幕宽度 | 适配策略 |
|------|----------|----------|
| 移动端 | < 768px | 单栏布局，底部固定导航 |
| 平板端 | 768px - 1024px | 双栏布局，导航栏显示 |
| 桌面端 | > 1024px | 多栏布局，全功能导航 |

## 5. 部署方式

| 平台 | 步骤 |
|------|------|
| **GitHub Pages** | 推送代码到仓库 → Settings → Pages → 选择分支 → Deploy |
| **Vercel** | 连接 GitHub 仓库 → 自动部署 → 获取访问地址 |
| **本地文件** | 直接双击 index.html 或通过本地服务器访问 |

---

## 6. 注意事项

- 图片统一使用 WebP 格式，减小文件体积。
- CSS 样式采用模块化设计，便于维护和扩展。
- 动画效果使用 transform 和 opacity，利用 GPU 加速。
- 遵循移动优先的响应式设计原则。
