# 产品需求文档

## 1. 项目概述

基于 HTML5 和 CSS3 开发的纯静态个人主页网站，采用温馨文艺视觉风格，包含封面页、个人主页、兴趣爱好、留言书屋和学习日志五大功能模块。

**设计风格**：柔和色彩、圆角设计、响应式布局、平滑过渡动画。

**目标用户**：大学生群体，热爱生活、喜欢记录、期望展示个人品味。

---

## 2. 用户角色

| 角色 | 需求描述 |
|------|----------|
| **普通访客** | 快速了解个人背景信息，浏览兴趣爱好展示，查看学习日志。 |
| **技术同行** | 查看技术栈和项目信息，通过联系方式进行技术交流。 |
| **潜在雇主** | 下载简历，了解工作经历和技能特长。 |
| **管理员（我）** | 通过修改 HTML/CSS 文件更新内容，无需编写 JavaScript 代码。 |

---

## 3. 核心功能模块

| 模块名称 | 功能描述 | 技术难点/亮点 |
|----------|----------|---------------|
| **封面页** | 炫彩视觉设计，展示个人姓名和学校信息，动态进入按钮引导跳转。 | CSS 多元素堆砌设计，悬浮动效。 |
| **个人主页** | 欢迎语、个人介绍、联系信息展示，侧边导航和底部固定导航。 | Flexbox 响应式布局，固定定位导航栏。 |
| **兴趣爱好** | 五个板块导航（电影、阅读、摄影、运动、心动时刻），支持查看、编辑、上传、删除、展开功能。 | 图片 hover 展开效果，CSS 动画过渡。 |
| **留言书屋** | 访客留言板、树屋居民登记表、活动日历展示。 | 表单样式设计，日历网格布局。 |
| **学习日志** | 日志列表展示、编辑、删除功能，支持图片上传和展开查看。 | 卡片式布局，交互反馈设计。 |

---

## 4. 非功能需求

| 需求类型 | 要求 |
|----------|------|
| **性能** | 首页加载时间 ≤ 3 秒，页面可交互时间 ≤ 5 秒。 |
| **兼容性** | 支持 Chrome、Firefox、Safari、Edge 最新两个版本。 |
| **响应式** | 适配移动端 (<768px)、平板端 (768-1024px)、桌面端 (>1024px)。 |
| **安全** | 输入内容转义处理，图片上传限制格式和大小。 |

---

## 5. 验收标准

- [ ] 所有页面正常加载，视觉与设计稿一致。
- [ ] 导航链接点击后正确跳转到对应页面。
- [ ] 兴趣爱好各板块功能正常（查看、编辑、上传、删除、展开）。
- [ ] 留言板和登记表表单验证正常。
- [ ] 响应式布局在三种屏幕尺寸下均正常显示。
- [ ] 动画效果流畅，无卡顿现象。
