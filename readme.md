# 个人主页展示

## 项目介绍
这是一个基于HTML5和CSS3开发的纯静态个人主页网站，采用温馨文艺的视觉风格，包含封面页、个人主页、兴趣爱好、留言书屋和学习日志等功能模块。

## 项目结构
```
.
├── index.html              # 封面页
├── readme.md              # 项目说明文档
├── css/                   # 样式文件目录
│   └── style.css          # 主样式文件
├── html/                  # 页面文件目录
│   ├── home.html          # 个人主页主页面
│   ├── hobbies.html       # 兴趣爱好页面
│   ├── message_room.html  # 留言书屋页面
│   └── learning_log.html  # 学习日志页面
└── img/                   # 图片资源目录
    ├── cover.jpg          # 封面图片
    └── ...                # 其他图片资源
```

## 功能说明

### 1. 封面页 (index.html)
- 展示网站入口页面，炫彩吸睛设计，多元素堆砌
- 显示个人姓名(文菲 fawn)和学校信息(武汉理工大学，中英双语)
- 欢迎词（忙里偷闲，爱健忘爱记录爱碎碎念的大学生~）
- 提供进入主页的引导按钮，带有动态效果

### 2. 个人主页主页面 (html/home.html)
- 欢迎语和个人介绍
- 联系信息展示
- 侧边栏目导航（兴趣爱好、留言书屋、学习日志）

### 3. 兴趣爱好页面 (html/hobbies.html)
- 提供五个兴趣爱好导航入口（电影、阅读、摄影、运动、心动时刻）
- 电影板块：展示喜欢的电影，支持编辑、上传、删除、展开功能
- 阅读板块：推荐喜欢的书籍，支持编辑、上传、删除、展开功能
- 摄影板块：展示摄影作品，支持图片上传、删除、展开功能
- 运动板块：展示运动爱好，支持图片上传、删除、展开功能
- 心动时刻：展示图片和文字，支持编辑、上传、删除、展开功能

### 4. 留言书屋页面 (html/message_room.html)
- 留言板：允许访问者留下留言和联系方式
- 树屋居民登记表：收集访问者个人信息和反馈
- 活动日历：展示2025年12月的日历

### 5. 学习日志页面 (html/learning_log.html)
- 后端文档编辑功能
- 支持日志的编辑、删除、修改
- 支持图片上传

## 技术栈
- **HTML5**：页面结构和内容
- **CSS3**：样式设计和页面布局
- **纯静态**：不使用JavaScript，所有效果通过CSS实现

## 设计风格
- 温馨文艺的视觉风格
- 柔和的色彩和圆角设计
- 响应式布局，适配不同屏幕尺寸
- 平滑的页面过渡和悬浮效果
- 图片展示支持鼠标悬停展开功能
- 底部固定导航栏，方便页面间切换

## 使用方法
1. 确保所有文件按照上述目录结构组织
2. 将所有图片资源保存在`img/`目录下
3. 直接在浏览器中打开`index.html`文件即可访问网站
4. 通过页面底部的导航栏可以在不同页面间切换

## 浏览器兼容性
支持所有现代浏览器（Chrome、Firefox、Safari、Edge等）

## 开发说明
- 所有HTML文件使用UTF-8编码
- CSS样式遵循模块化设计原则
- 图片资源建议使用WebP格式，以提高加载速度
- 响应式设计断点：
  - 移动端：< 768px
  - 平板：768px - 1024px
  - 桌面端：> 1024px
